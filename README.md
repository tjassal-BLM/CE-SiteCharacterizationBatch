# CE-SiteCharacterizationBatch

Instructions

1. Clone the repository on your local machine.
2. Open the R project file (.Rproj).
3. See detailed instructions in the R script file (.R) to run a reproducible example.
4. Note: you will need to request a free API key from Climate Engine to run the code (https://support.climateengine.org/article/36-requesting-an-authorization-key-token)
5. The R markdown file (.Rmd) is a duplicate of the .R file; included for preference.
